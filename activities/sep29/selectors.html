What is the difference between an ID and a Class?

An ID is used to identify one specific element on a page, while a class can be used to identify multiple elements on a page.

What is the difference between a SPAN tag and a DIV tag?

A SPAN tag is called an inline element, it stlects text in a line to add decoration to. A DIV tag creates a block and adds line breaks as well.
