Beginning CSS

Explain the difference between inline, embedded, and external CSS.

inline: The css is right next to the html code its affecting
embedded: The css code is in the same file as the html code it is affecting.
external: The css code is in a different file than the html code it is affecting.

Explain why you should only use external CSS.
This is because it is the easeast way to navigate the css code and makes updating it easier. Also then others can write the css code and you can write the html code.

What is a selector used for?
It is used to chose what items on a page get effected.It goes at the start of the css block.
