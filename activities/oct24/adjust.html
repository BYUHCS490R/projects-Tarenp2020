Why is the viewport meta tag important?

This tells the screen to keep the scall and make the user zoom in or to let the scale adjust how its been set to.

How can you adjust the CSS for printing?

You make a new CSS file and you link it at the top with the the media tag being "print" and the normal css being "screen".

When would you want to adjust the CSS for printing?

When you know there is a reason someone would want to print like a coupon or a QR code.
